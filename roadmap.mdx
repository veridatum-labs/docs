title: "Roadmap"
description: "EarnProof delivery path from testnet MVP to protocol ecosystem"
---

EarnProof starts with a narrow testnet workflow and expands toward issuer-backed credentials, SDK integrations, and stronger privacy mechanisms.

## Foundation

- Public repositories
- Local setup documentation
- Frontend app shell
- Backend API shell
- Soroban contract workspace
- Contributor-ready issues

## Wallet and Payment Indexing

- Freighter wallet connection
- Signed wallet challenge authentication
- Stellar testnet payment sync
- Payment normalization
- Payment classification
- Trusted sources

## Proof Engine

- Minimum income proof
- Recurring income proof
- Payment receipt proof
- Deterministic credential payloads
- Credential signing
- Credential verification

## Public Verification

- Public verification page
- Machine-readable verification endpoint
- Valid, expired, revoked, invalid, and unknown proof states
- Verification event logging with privacy controls

## Soroban Trust Layer

- Protocol configuration contract
- Issuer registry contract
- Proof registry contract
- Testnet deployment scripts
- Backend integration adapters
- Contract event indexing

## Issuer Features

- Organization registration
- Issuer application review
- Issuer-backed attestations
- Issuer revocation
- Issuer directory

## Developer Platform

- API keys
- Webhooks
- OpenAPI documentation
- TypeScript SDK
- React verification badge
- Example integrations

## Future Privacy Work

- Range proofs
- Greater-than proofs
- Aggregate proofs
- Confidential token support when production-ready
- Standards-based verifiable credential interoperability
